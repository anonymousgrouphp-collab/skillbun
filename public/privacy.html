Privacy Policy

Last updated: February 26, 2026

1. Information You Provide

During onboarding, SkillBun may collect your name, email, degree/program, and current year. This profile is used to personalize quiz and counsellor interactions.

2. Local Storage on Your Device

We store profile fields in your browser localStorage (`sb_name`, `sb_email`, `sb_degree`, `sb_year`) so you can continue using the app without re-entering details every time. For Bun-Bot chat, a local rate-limit counter (`sb_counsel_rl`) is also stored on your device.

3. Server-Side Profile Storage (Optional)

If server storage is configured, your profile may be synced to a Supabase table. Stored fields can include name, email, degree, year, entrypoint, a hashed IP value, and a truncated user-agent string. If Supabase is not configured, profile sync is skipped.

4. AI Processing

Quiz responses and counsellor chat prompts are sent to the SkillBun backend, which forwards requests to the Google Gemini API to generate responses. Please avoid submitting sensitive personal data in free-text prompts.

5. Bot and Abuse Protection

We use human-verification controls, including Cloudflare Turnstile when enabled, and signed short-lived human-proof tokens to protect APIs from abuse.

6. Cookies and Tracking

SkillBun does not currently run ad tracking pixels or analytics trackers in this app version. Core functionality relies mainly on browser localStorage and essential network logs.

7. Data Retention and Deletion

Local profile data can be removed by logging out or clearing browser storage. If your profile has been synced to server storage, contact us to request deletion or correction.

8. Security

We apply technical safeguards including request size limits, input validation, rate limiting, and security headers. No online system can be guaranteed fully secure, but we continuously improve controls.

9. Changes to This Policy

This policy may be updated as features evolve. We will publish the latest version on this page with a revised date.
